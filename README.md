# 🚗 BMW Global Sales Dashboard | Power BI

## 📌 Project Overview
Developed an interactive Power BI dashboard analyzing BMW car sales performance across multiple countries, models, and sales channels.

The dashboard provides insights into:
- Revenue & Revenue PY comparison
- Year, Month & Weekday performance analysis
- Top Selling Models
- Most Expensive Models
- Sales by Country
- Sales by Channel (Online, Dealership, Wholesale)
- Drill-through model-level analysis

---

## 📊 Key Features

✅ Dynamic KPI Cards (Revenue, PY Revenue, Growth %)  
✅ Time Intelligence (YoY Comparison)  
✅ Drill-through Model Detail Pages  
✅ Interactive Period Slicer  
✅ Custom Model Image Integration  
✅ Country-wise Sales Analysis with Flags  
✅ Channel-wise Sales Distribution  

---

## 🛠 Tools Used

- Power BI Desktop
- DAX (Time Intelligence Functions)
- Data Modeling (Star Schema)
- Custom Image URL Integration
- Drill Through & Bookmarks
